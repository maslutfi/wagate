Common Questions

Everything you need to know about WAGate.

Will my WhatsApp number get banned?

WAGate uses the official Go-WhatsApp library which mimics the browser session. Respect WhatsApp's Terms of Service: don't send unsolicited spam, and maintain a reasonable delay between messages. Our Pro version includes built-in rate-limiting tools to help you stay safe.

How does self-hosting work?

WAGate is distributed as a lightweight Docker container or a standalone binary. You can run it on any VPS (even a $5/mo server), home server, or Raspberry Pi. Once running, you scan a QR code just like WhatsApp Web, and your API is ready.

Does it support n8n community node?

Yes! We have a dedicated n8n node that allows you to drag-and-drop WAGate into your workflows. No complex HTTP Request nodes neededâ€”just enter your WAGate URL and API Key.

Can I upgrade from Free to Pro later?

Absolutely. You can start with the Free tier to test your integration. When you're ready for bulk features or unlimited devices, simply purchase a license and enter the key in your dashboard. No data migration required.

Is my data secure?

WAGate is 100% self-hosted. Your message history, session keys, and media never leave your server. We don't have a middle-man server, making it much more secure than cloud-based WhatsApp APIs.
